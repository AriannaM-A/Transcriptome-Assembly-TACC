# Data 
This is a collection of the data used to assemble this transcriptome. The samples were not randomly chosen and were all used in the assembly of this transcriptome.

This project uses 111 RNA-Seq accessions sourced from multiple laboratories under varying conditions. All samples included in the analysis are catalogued in the accompanying metadata file. The table records the accession ID, sequencing type, project identifiers, sample provider, and any known collection or growth conditions.
